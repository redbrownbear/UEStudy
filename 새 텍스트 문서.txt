새로운 레벨(맵)을 만들때

1. GameMode 생성
 - 블프로 GameMode 생성
 - Level에 연결

2. 아이템 생성
 - 블프로 아이템 생성
 - 스켈레탈 메시 컴포넌트 연결(메시마다 다른가?)
 - 이벤트그래프에서 회전 (아이템이 여기 있다라는 효과)
 - Add Actor Local Rotation을 EventTick에 연결

3. HUD 추가
 - 블프로 HUD 추가
 - Create UI (놓침...)

 - UI에서 캔버스 패널 추가
 - 보더 추가(가운데 정렬 -> 크로스해어를 위함)

캐릭터와 보더가 겹치므로 캐릭터 위치 조정요함

4. Character추가.. 이거 어떻게 하누.. 복붙인가?
 -상속받아서 추가하는듯

5. HUD에서 크로스해어를 분리
 - UI를 하나 추가해서 크로스해어용으로 만든다.
 - 캔버스 패널 -> 보더 순으로 만든다.

6. 텍스트를 표기를 위한 UI 하나 추가

7. 분리 후 부모 HUD에서 모든 자식 UI를 관리한다
 - 크로스해어, 텍스트용 UI를 하위에 넣어준다.

8. 트리거를 이용 아이템에 캐릭터가 오버랩 되면 
    텍스트를 넣어준다(예 : 어떤 키를 입력하면 획득 할 수 있습니다.)

9. 커스텀 함수(블프)를 만들어서 오버랩 될때, 빠질때
    기능을 만들어 준다

10. 컨트롤러 추가
 - 상속받아서 기존에 만들었던 컨트롤러를 상속받아 사용
